# data-processor

A simple data processor for the [DesInventar](www.desinventar.net) and [EM-DAT](https://www.emdat.be/).

## Installation

```bash
git clone https://github.com/COMP0016-IFRC-Team5/data-processor.git
cd data-processor
```

## Usage

```bash
python3 main.py
```
